## Tensor Dimension & Shape Practice (Frontend Web App)

This simple frontend app presents 10 PyTorch tensor snippets. Students decide each tensor's dimension (rank) and shape, then click Reveal Answer to check. No submission is required.

### Run locally

Open `index.html` directly in a browser, or serve the folder locally (recommended to avoid any browser restrictions with local files):

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

### Notes
- Tensors include examples from dimension 0 through 4.
- Each card shows the snippet and lets you reveal/hide the answer individually, or use the global Reveal All / Hide All controls.
